Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web.

  • HTML is a markup language, not a programming language, meaning it annotates text to define how it is structured and displayed by web browsers.

  • It forms the building blocks of all websites and is complemented by CSS for style and JavaScript for interactivity.

In a nutshell, HTML is all about organizing and displaying information on a webpage. We can think of it as the bones or structure of a webpage

Here we gonna make some simple profile web page by using HTML and CSS

Head

Before we go to Head we're gonna make basic structure of html

          
            <!DOCTYPE html>
            <html lang="en">
            </html>

  • <!DOCTYPE html> : Declares the document type and version of HTML (HTML5 in this case).

  • <html> : he root element that wraps all the content on the page.

Now we need to add head to out html to add title element, by simply add head we now can add meta to our html, meta is term for metadata metadata contain the description of our program.

<head>: Contains meta-information about the page, such as the title, character set, and links to stylesheets.

Now we add title of your profile web by adding title element, and we also need to add charset attribute with meta element, we also need to add link to style your page using CSS (we gonna talk about this later)

            <html lang="en">
            <head>
              <title>My Profile Web</title>

            </html>

Header

In HTML, the <header> tag and heading tags (<h1> to <h6>) serve different but complementary purposes.

The <header> tag is a semantic element used to define the introductory or navigational content of a webpage or a section of a webpage. It typically contains elements like logos, navigation menus, or headings.

Headings are used to define the titles and subtitles of sections in a webpage. They range from <h1> (the most important) to <h6> (the least important).

Styles

Remember the link to styles.css that we have added before in head element now we gonna make a styles.css in same directory as your html file

After you make the file we now ready to style the page, first we gonna introduce you to selector in CSS there's 3 type of selector that we gonna use.

element selector targets HTML elements based on their tag name (e.g., <p>, <h1>, div>).

          
              element {
              property: value;
              }

class selector targets elements that have a specific class attribute. Classes are reusable and can be applied to multiple elements. Starts with a . (dot) followed by the class name.

          
              .class-name {
                property: value;
              }

ID selector targets a single element with a specific id attribute. IDs are unique and should only be used once per page. Starts with a # (hash) followed by the ID name.

          
              #id-name {
                property: value;
              }

Now after we know we can use selector to select spesific html element, lets put those into our html file so we can start styling our page.
Your html file should like this.

          
            <!DOCTYPE html>
            <html lang="en">
            <head>
              <title >My Profile Web</title>

              <link rel="stylesheet" href="styles.css"/>

Profile style

Now we get to main page of our website, we want to make the background profile and education history to be side by side, to do that we first need to make the display attribute value set to flex in the profile class, after that we need to make a space between profile text and window border on the right side of the screen, we need to add padding. after that we want to justify our text by using text-align and set it's value to justify

          
            .profile{
              display: flex;
              margin-bottom: 20px;
            }
            .background-profile{
              max-width: 55%;
              padding: 0 20px 0 20px;
            }

            .profile-text{
              text-align: justify;
              text-justify: inter-word;
              font-size: 1.1em;
            }

            .education{
              max-width: 35%;
              font-weight: 100;
            }

See that I add max-width in each div, it's initialize that the max width of background profile are fill the 55% of the page, and so the education history can have a space to get side by side with the profile background
